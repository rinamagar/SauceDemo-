 # SauceDemo
This project contains manual test cases for the SauceDemo website.

Modules Covered:
  -login page
  -Signup page
  -Dashboard page
  -Checkout Page

Testing Type:
  -Functional testing
  -Positive and Negative Testing
  -UI Validation

Test cases include: 
  -Test case ID
  -Test Description/title
  -Preconditions
  -Test Steps
  -Test data
  -Expected Result
  -Actual Result
  -Status

Objective: 
-> validate user authentication functionality
-> Verify product selection and cart behavior
-> Validate checkout process
-> Identify functional defects
Tools used:
  Google Sheets
  Chrome browser/microsoft edge
