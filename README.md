 # SauceDemo
This project contains manual test cases for the SauceDemo website.

Modules Covered:
<br>
<ul>
 <li> login page</li>
 <li> Signup page</li>
 <li> Dashboard page</li>
 <li> Checkout page</li>
</ul>
 

Testing Type:
<ul>
 <li>Manual testing</li>
 <li>Functional testing</li>
 <li>UI testing</li>
 <li>Positive & Negative testing </li>
</ul>

Test cases include: 
<ol>
 <li>Test case ID</li>
 <li>Test Description</li>
 <li>Preconditions</li>
 <li>Test Steps</li>
 <li>Test Data</li>
 <li>Expected Result</li>
 <li>Actual Result</li>
 <li>Status</li>
</ol>
  

Objective: 
<ul>
 <li>validate user authentication functionality</li>
 <li>Verify product selection and cart behavior</li>
 <li>Validate checkout process</li>
 <li>Identify functional defects</li>
</ul>

Tools used:
<ul>
 <li>Google Sheets</li>
 <li>Chrome browser/microsoft edge</li>
</ul>
